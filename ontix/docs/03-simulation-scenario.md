### 互動數據追蹤 (Engagement Snapshots)

由於互動數據（轉發、評論、按讚）會隨時間持續增長，系統採用 **快照機制** 記錄變化：

│                                                                             │

│  ──────────────────┼───────┼───────┼────────┼───────┼──────────────────────  │
│  T+0   14:23:00   │    0  │    0  │     0  │    0  │ -                      │
│  T+5   14:28:00   │   23  │   45  │   156  │   34  │ 🔥 4.6次/min          │
│  T+10  14:33:00   │   89  │  156  │   567  │   89  │ 🔥🔥 13.2次/min       │
│  T+15  14:38:00   │  234  │  389  │  1,247 │  178  │ 🔥🔥🔥 29次/min       │

**分類型增長曲線**:
```

  │

  │                                         ●
  │                                    ●
4K├                                              ▲ 轉發 (4,567)
  │                                         ▲
  │
3K├                               ●     ▲
  │                          ●    ▲
  │
2K├                     ●    ▲
  │                ●    ▲
  │           ●    ▲
1K├      ●    ▲
  │ ●    ▲
  │▲●
  └──┬────┬────┬────┬────┬────┬────┬────────▶ 時間
    T+0  T+15  T+30  T+1h T+2h T+3h T+6h

  ▲ 轉發數 - 傳播擴散主力，跨平台風險高
  ● 評論數 - 輿論發酵指標，負面情緒放大器
```

**互動類型分析**:
- 此案例為 **評論驅動型** 傳播（評論數 > 轉發數）
- 評論區可能出現大量跟風吐槽，需優先監控評論情緒
- 若為 **轉發驅動型**，則重點關注跨平台擴散風險

**快照策略**:
- **即時監控** (is_key_post = true): 前 2 小時每 5 分鐘記錄一次
- **一般監控**: 每小時記錄一次
- **冷卻期**: 增長速度 < 1次/min 後，降為每日記錄

**不同互動類型的風險權重**:

| 互動類型 | 風險權重 | 說明 |
|----------|----------|------|
| **轉發** | ⭐⭐⭐⭐⭐ | 最高風險：擴大傳播範圍，跨平台擴散主要途徑 |
| **評論** | ⭐⭐⭐⭐ | 高風險：反映輿論態度，可能出現負面跟風評論 |
| **按讚** | ⭐⭐ | 中等：表示認同，但不主動擴散 |
| **收藏** | ⭐ | 低風險：私人行為，不影響傳播 |

**複合指標計算**:
```
風險傳播分 = (轉發數 × 5) + (評論數 × 3) + (按讚數 × 1)
傳播速度 = Δ風險傳播分 / 時間間隔(分鐘)
```

**快照用途**:
1. 計算傳播速度，用於風險評估
2. 繪製增長曲線，判斷是否達到峰值
3. 評估回應後的互動變化（效果衡量）
4. 異常檢測（突然爆發或人為刷量）
5. 分析哪種互動類型主導傳播（轉發驅動 vs 評論驅動）
